Adding blocks to PyCraft

This tutorial shows how to add blocks to PyCraft. It's as easy as it sounds!

Posted by somePythonProgrammer on January 1st, 2022

PyCraft is a minecraft clone written in python. Adding blocks to it is fairly easy, if you don't mind the texturing part.

This tutorial is for those who are fairly new to Coding.

First, we need to fork and clone the repository. We'll use the git command to clone the repository.

Cloning the repository is as simple as typing the following:

git clone https://github.com//PyCraft.git

Replace USERNAME with your github username.

Now, we'll need to go into the directory we just cloned the repository to. We'll use the cd command to go into the directory. We'll go into the directory by typing the following:

cd PyCraft

Now, we'll need to install the dependencies. We'll use the dep_installer batchfile to install the dependencies. To run the batchfile, simply double-click it.

Next, we'll try running PyCraft. We'll use the __main__.py file to run PyCraft. To run the file, open the terminal in the project directory, and type the following:

python __main__.py

Now, we'll add a block to the game. Close PyCraft, if you already haven't.

First of all, you need to make a texture. Here's a fairly new texture recently added to the game:

Please note that the texture extension must be *.png, and the texture must be sized to 32x32 pixels.

Now, we'll need to add the texture to the game. To do this, we'll paste the images in the /assets/textures/block/ directory. Now, we are ready to add the code for the block.

To add the code, we need to make the file for our block in the /Classes/terrain/blocks/ directory. Please note that the name of the file needs to be the same as the name of the block.

We'll make a file called Grout.py for an example. Please note that the code is already there in the repository, so you can make another block. The texture I showed you earlier was made for the same block. It's file is grout_sand.png.

Open the PyCraft folder in the text editor of your choice. I use VS Code. Next, make and open the file: Classes/terrain/blocks/Grout.py and paste the template code into it:

import Classes as pycraft

class Block(pycraft.Block):
    def __init__(self, *args, **kwargs):
        """
        Name
        
        * Initializes the block
        
        :parent: the parent window
        """
        super().__init__("Name", *args, **kwargs)

        self.texture = {
            "top": self.parent.textures["texture_name"],
            "front": self.parent.textures["texture_name"],
            "back": self.parent.textures["texture_name"],
            "left": self.parent.textures["texture_name"],
            "right": self.parent.textures["texture_name"],
            "bottom": self.parent.textures["texture_name"]
        }

As this is only the template, we are going to replace 'Name' in the file with the filename/blockname. In this case it will be Grout:

Now, we need to add the textures to our block. For this, we need to replace texture_name with the names of the textures. In this case, they will all be grout_sand as our image is grout_sand.png.

import Classes as pycraft

class Block(pycraft.Block):
    def __init__(self, *args, **kwargs):
        """
        Grout
        
        * Initializes the block
        
        :parent: the parent window
        """
        super().__init__("Grout", *args, **kwargs)

        self.texture = {
            "top": self.parent.textures["grout_sand"],
            "front": self.parent.textures["grout_sand"],
            "back": self.parent.textures["grout_sand"],
            "left": self.parent.textures["grout_sand"],
            "right": self.parent.textures["grout_sand"],
            "bottom": self.parent.textures["grout_sand"]
        }

Well done! We've added our block! We have to do nothing else as the game automatically does the rest for us!

Now, you can run PyCraft, and scroll to see the current block name in the bottom left. Scroll until you see your block. Try placing it by pointing at a block and right-clicking.

Now, commit and push: 

git add .
git commit -m "Added Block"
git push

Then, open a pull request on github, and we'll check it out. If we like it, we'll merge the pull request. If you'd like to join our community, open a discussion on the repository: 

https://github.com/1upCommunity/Contact-Us/discussions

Happy coding!
